## 🚀 Déployer sur Vercel (5 min)

1. Allez sur **https://vercel.com**
2. Sign Up avec GitHub
3. "Import Project"
4. Collez: `https://github.com/wangfed2007-oss/freefire-shop`
5. Cliquez "Deploy"
6. **BOOM! C'est live!** 🎉

Voir **VERCEL_DEPLOYMENT.md** pour plus de détails.
