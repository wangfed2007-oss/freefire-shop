# 🚀 Déploiement Vercel - Free Fire Shop

## Option 1: Déploiement Automatique (Recommandé ⭐)

### Étape 1: Aller sur Vercel
1. Allez sur https://vercel.com
2. Cliquez **"Sign Up"**
3. Choisissez **"Continue with GitHub"**
4. Autorisez Vercel à accéder à vos repos

### Étape 2: Importer le Projet
1. Une fois connecté, cliquez **"Import Project"** ou **"New Project"**
2. Collez l'URL du repository:
   ```
   https://github.com/wangfed2007-oss/freefire-shop
   ```
3. Cliquez **"Import"**

### Étape 3: Configuration Automatique
Vercel détecte automatiquement:
- ✅ Framework: **Next.js**
- ✅ Build Command: `npm run build`
- ✅ Output Directory: `.next`
- ✅ Install Command: `npm install`

**Cliquez juste "Deploy"** - C'est tout! 🎉

---

## Étape 4: Votre Site est LIVE! 🌍

Après 30-60 secondes, vous recevez une URL:
```
https://freefire-shop-[random].vercel.app
```

**Accédez à votre application:**
- 🏠 Page d'accueil: `https://votre-site.vercel.app/`
- 😂 Blagues: `https://votre-site.vercel.app/jokes`
- 🎉 API Playground: `https://votre-site.vercel.app/api-playground`

---

## Option 2: Domaine Personnalisé Gratuit

### Ajouter un sous-domaine gratuit .vercel.app:

1. Dans Vercel Dashboard → Votre projet
2. Allez dans **Settings**
3. Cliquez **"Domains"**
4. Entrez un nom personnalisé:
   ```
   freefire-shop.vercel.app
   ```
5. Cliquez **"Add"**

**Voilà! Votre site est accessible à:**
```
https://freefire-shop.vercel.app
```

---

## ✅ Redéploiement Automatique

Chaque fois que vous faites un `git push`:
1. GitHub notifie Vercel
2. Vercel reconstruit le projet
3. Votre site se redéploie automatiquement
4. **Zéro configuration!** 🚀

```bash
# Exemple:
git add .
git commit -m "Update content"
git push origin main
# → Vercel redéploie automatiquement!
```
